# Types of Cybersecurity Threats

Cyber threats wo activities hoti hain jo systems, networks
ya data ko nuksan pohanchane ke liye ki jati hain.

## 1. Malware

Malware ka matlab hota hai malicious software jo system ko harm karta hai.

Types of Malware:
- Virus – files ko infect karta hai
- Worm – khud spread hota hai network mein
- Trojan – fake software ban kar attack karta hai
- Ransomware – data lock karke paisay maangta hai

Example:
Aisa software jo PC slow kar de ya files delete kar de.

## 2. Phishing

Phishing ek fake message ya email hoti hai jo user se
password ya personal information churaane ke liye bheji jati hai.

Example:
Fake bank email jo kehta hai “Apna password verify karo”.

## 3. Brute Force Attack

Is attack mein hacker bohat saare passwords try karta hai
jab tak correct password na mil jaye.

Example:
Login page par thousands passwords automatically try karna.

## 4. DoS / DDoS Attack

DoS (Denial of Service):
Server ko itni zyada requests bhejna ke wo crash ho jaye.

DDoS (Distributed Denial of Service):
Bohat saare systems mil kar ek server par attack karte hain.

Example:
Website ka down ho jana.

## Key Takeaways

- Malware system ko directly harm karta hai
- Phishing users ko trick karke data churaati hai
- Brute force passwords guess karta hai
- DDoS website ya server ko unavailable bana deta hai
