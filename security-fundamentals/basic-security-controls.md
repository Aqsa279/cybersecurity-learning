# Basic Security Controls

Security controls wo tools aur techniques hain jo systems aur data ko protect karte hain.

## 1. Firewall

Firewall ek network security system hai jo incoming aur outgoing traffic ko monitor karta hai
aur unauthorized access ko block karta hai.

Example:
- Office network ka firewall sirf authorized users ko allow karta hai

## 2. Antivirus

Antivirus software malware ko detect aur remove karta hai.

Example:
- PC ya mobile par antivirus malware scan karke delete karta hai

## 3. IDS / IPS

- IDS (Intrusion Detection System): Suspicious activity detect karta hai
- IPS (Intrusion Prevention System): Suspicious activity detect + block karta hai

Example:
- Network mein unusual login attempts ko alert ya block karna

## 4. Encryption

Encryption data ko unreadable form mein convert karta hai
taake unauthorized log usay samajh na saken.

Example:
- HTTPS websites ka data secure rehta hai

## 5. Backup

Backup ka matlab hai data ka duplicate copy banana
taake data loss ke case mein restore kiya ja sake.

Example:
- Important files ka cloud backup

## Key Takeaways

- Firewalls aur antivirus system ko protect karte hain
- IDS/IPS suspicious activity detect aur prevent karte hain
- Encryption data ko secure banata hai
- Backup data loss se bachata hai
