# Authentication vs Authorization

Authentication aur Authorization cybersecurity ke
do important security concepts hain.

## 1. Authentication

Authentication ka matlab hota hai user ki identity verify karna.

Simple words mein:
"Tum kaun ho?"

Examples:
- Username & password
- OTP
- Fingerprint / Face ID

Real-life example:
ATM card + PIN dalna = Authentication

## 2. Authorization

Authorization ka matlab hota hai user ko permissions dena.

Simple words mein:
"Tum kya kya kar sakte ho?"

Examples:
- Admin access
- Read-only access
- File permissions

Real-life example:
ATM se sirf apna paisa nikal sakte ho, doosron ka nahi.

## Authentication vs Authorization (Difference)

Authentication:
- Identity check karta hai
- Login se pehle hota hai

Authorization:
- Access level check karta hai
- Login ke baad hota hai

## Key Takeaways

- Authentication batata hai ke user kaun hai
- Authorization batata hai ke user kya kar sakta hai
- Dono mil kar system ko secure banate hain
