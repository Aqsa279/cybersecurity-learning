# Ethical Hacking Overview

Ethical Hacking ek practice hai jisme systems, networks
aur applications ko test karke security flaws find kiye jate hain.

## What is Ethical Hacking?

- Legal hacking jo permission ke sath ki jati hai
- Security improve karne ke liye flaws find karna

## Types of Hackers

- **White Hat:** Legal hackers, security improve karte hain
- **Black Hat:** Illegal hackers, damage ya data steal karte hain
- **Grey Hat:** Kabhi legal, kabhi illegal, mostly testing

## Why Ethical Hacking is Important

- Systems aur data ko secure banata hai
- Vulnerabilities ko time pe patch karna easy hota hai
- Companies aur users ka trust build hota hai
- Cyber attacks prevent hote hain

## Bug Bounty Programs

- Companies reward deti hain security researchers ko
- Flaws report karne par prize ya recognition milta hai
- Example: HackerOne, Bugcrowd

## Key Takeaways

- Ethical hacking legal aur permission-based hacking hai
- White hat hackers security improve karte hain
- Bug bounty programs beginners ke liye practice aur rewards ka source hain
