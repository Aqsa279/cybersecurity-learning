# CIA Triad in Cybersecurity

CIA Triad cybersecurity ka basic security model hai jo data ko protect
karne ke liye use hota hai.

CIA ka matlab hai:
- Confidentiality
- Integrity
- Availability

## 1. Confidentiality

Confidentiality ka matlab hai ke data sirf authorized log hi access kar saken.

Examples:
- Password protection
- Data encryption
- User authentication

Real-life example:
Bank account details sirf account holder hi dekh sakta hai.

## 2. Integrity

Integrity ka matlab hai ke data change na ho jab tak authorized person na kare.

Examples:
- Hashing (MD5, SHA)
- Checksums
- Digital signatures

Real-life example:
Exam result ya transaction amount ka change hona detect ho jana.

## 3. Availability

Availability ka matlab hai ke data aur systems hamesha available hon jab zaroorat ho.

Examples:
- Server backups
- Load balancing
- Protection against DDoS attacks

Real-life example:
Bank ATM ya website ka hamesha accessible rehna.

## Why CIA Triad is Important

- Security policies CIA Triad par based hoti hain
- Risk analysis mein help milti hai
- Attacks ko samajhna asaan hota hai

## Key Takeaways

- Confidentiality data ko private rakhti hai
- Integrity data ki accuracy ensure karti hai
- Availability systems ko accessible rakhti hai
